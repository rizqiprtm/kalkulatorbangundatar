Luas Persegi

Rumus Luas Persegi yaitu :

L = S x S

Di mana :
L = Luas
S = Sisi

K = 4 x S

Di mana :
K = Keliling

Luas Persegi Panjang

Rumus Luas Persegi Panjang yaitu :

L = P x L

Di mana :
L = Luas
P = Panjang
L = Lebar

Hitung Luas Persegi Panjang



Keliling Persegi Panjang

Rumus Keliling Persegi Panjang yaitu :

K = 2 (P + L)

Di mana :
K = Keliling
P = Panjang
L = Lebar

Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x A x t

Di mana :
L = Luas
A = Panjang Alas
t = Tinggi

Hitung Luas Segitiga



Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Di mana :
K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C

Hitung Keliling Segitiga

L = Luas

t = Tinggi

Hitung Luas Jajar Genjang



Keliling Jajar Genjang

Luas Trapesium

Rumus Luas Trapesium yaitu :

L = 1/2 x (S1 + S2) x t

Di mana :
L = Luas
S1 = Sisi A
S2 = Sisi B
t = Tinggi

Hitung Luas Trapesium




Keliling Trapesium

Rumus Keliling Trapesium yaitu :

K = S1 + S2 + S3 + S4

Di mana :
K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C
S4 = Sisi D

Hitung Keliling Trapesium





Luas Layang-Layang

Rumus Luas Layang-Layang yaitu :

L = 1/2 x d1 x d2

Di mana :
L = Luas
d1 = Diagonal Vertikal
d2 = Diagonal Horizontal

Hitung Luas Layang-Layang



Keliling Layang-Layang

Rumus Keliling Layang-Layang yaitu :

K = 2 x (S1 + S2)

Di mana :
K = Keliling
S1 = Sisi 1
S2 = Sisi 2

Hitung Keliling Layang-Layang

Luas Lingkaran

Rumus Luas Lingkaran yaitu :

L = π x r x r

Di mana :
L = Luas
π = Phi (3,14 atau 22/7)
r = Jari-Jari Lingkaran

Hitung Luas Lingkaran


Keliling Lingkaran

Rumus Keliling Lingkaran yaitu :

L = π x d

Di mana :
L = Luas
π = Phi (3,14 atau 22/7)
d = Diameter Lingkaran
